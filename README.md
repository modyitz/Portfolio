# Portfolio Project — Setup Log

This repository documents the first step: installing the required tools, setting up a public GitHub repository, and recording the process. Below is a plain account of what I installed, the steps I completed.

## What tools I installed

- **Cursor IDE** — the code editor used for the whole process, installed from https://cursor.com/
- **Claude Code add-on** — added through Cursor's Extensions panel (search "Claude Code") and logged in.
- **Codex add-on** — added through Cursor's Extensions panel (search "Codex") and logged in.
- **Git / GitHub** — created/used a GitHub account at https://github.com/ to host this public repository.

## What steps I completed

1. Installed Cursor IDE from the official site.
2. Opened the Extensions panel in Cursor (`Cmd+Shift+X` on Mac) and installed the **Claude Code** add-on, then signed in.
3. Searched for and installed the **Codex** add-on the same way, then signed in.
4. Created a **public GitHub repository** to hold this project.
5. Opened the repository in Cursor.
6. Created this **README.md** file describing the tools, the steps, and the issues I worked through.

## What issues I ran into and how I solved them

The installation itself went smoothly Cursor, both add-ons, and the GitHub account were all set up without errors. I'd done each step correctly:

- **Finding the add-ons in Cursor.** I wasn't sure at first where extensions were managed. I solved this by opening the Extensions panel (`Cmd+Shift+X`), searching by name, and confirming I installed the official versions before logging in.
- **Confirming each add-on was actually signed in.** After installing, I checked that I was logged in to each tool rather than assuming the install was enough, so both Claude Code and Codex were ready to use.
- **Making sure the repository was public.** When creating the GitHub repo I double-checked the visibility setting was set to *Public* (not Private) so the README link would be openable by the reviewer.
- **Getting oriented in Cursor.** Since the editor was new to me, I spent a little time learning the layout — the file explorer, the editor pane, and the Markdown preview — before writing this file, using Cursor's preview to confirm the README rendered correctly.

Where I was unsure of a step, I looked it up rather than guessing, which is how I confirmed the extension install flow and the repo visibility setting.

*Tools used: Cursor IDE, Claude Code, Codex, Git, GitHub.*
